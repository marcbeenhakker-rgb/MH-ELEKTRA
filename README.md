# MH-Elektra (Next.js)

## Run lokaal
```bash
npm install
npm run dev
```

## Deploy (Vercel)
- Push naar GitHub
- Import in Vercel
- Deploy

## Logo & slider foto's
- Logo: `public/logo.png`
- Slider foto's: `public/slides/slide1.jpg`, `slide2.jpg`, `slide3.jpg`

## Extra logo’s (merken/partners) zonder gedoe
1. Zet je logo-bestanden in `public/logos/` (png/jpg/jpeg/webp/svg)
2. Refresh de site: ze verschijnen automatisch op de homepage onder **“Logo’s / merken”**
